mkdir folder_name = Used to create a directory/folder
cd folder_name = Change directory/Folder
touch file.txt = Create a empty file
cat file.txt = Displays content of the file.txt
vi file.txt = Opens the VI editor to manipulate the file content
>>>> Shift + i = Enables insertions in VI editor
>>>> ESC -> :wq = Quite and save the changes in the file.
ls : shows the list of files and directoy
cd .. : Change to dirctory to previous parent directory
